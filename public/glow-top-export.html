Theme thumbnail · Glow · Top

Deterministic, no-LLM. Each card below is the same template fed a different config — only the 4 palette colours, the 2 fonts, the surface (dark/light) and the name change. Resize the window to see the container-query type scale.

Surface → mode

surface luminance < 0.42 ⇒ dark (black field, white text). Otherwise light (white field, black text).

Bloom colours

The 2 most-saturated of primary/secondary/tertiary/accent become the bloom’s core → ring. Neutrals are skipped.

Fonts

titleFont sets the name, bodyFont the caption. Google fonts auto-load; system fonts used as-is.

To add a theme

Push a config to THEMES. No regeneration — colours & fonts swap into CSS variables.
